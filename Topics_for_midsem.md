1. Linear Data Structures
   * Arrays & Matrices
       * Multi-dimensional Array operations (Matrix Addition, Subtraction, Multiplication, Transpose).
       * Matrix Traversal (e.g., Spiral Matrix).
       * Array manipulations: Shifting, Rotation, and Dynamic Insertion/Deletion.
   * Linked Lists
       * Singly Linked List implementation (Node structure and Pointers).
       * Operations: Insertion (Start, End, Before/After element), Deletion (Start, End, Specific value), and Node Counting.
   * Stacks
       * Implementation using Arrays and Structures.
       * Applications: Infix to Postfix conversion, Postfix expression evaluation.
   * Queues
       * Linear Queue (Basic implementation).
       * Circular Queue (Efficient space utilization).

  2. Sorting Algorithms
   * Comparison-based Sorts: Bubble Sort, Insertion Sort, Selection Sort.


  3. Searching Algorithms
   * Linear Search: Sequential scanning.
   * Binary Search: Logarithmic search on sorted data (Iterative/Recursive).


  4. Recursion & Algorithmic Techniques
   * Types of Recursion: Tail Recursion and Non-Tail Recursion (NTR).
   * Mathematical Recursion: Ackermann Function, GCD (HCF) and LCM calculation.
   * Problem Solving: Tower of Hanoi, Recursive Array/String Reversal.
